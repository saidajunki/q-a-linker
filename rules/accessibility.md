# アクセシビリティ（a11y）

## 本ファイルの方針

WCAG準拠の詳細なガイドラインは、実装時に以下を参照すること：
- WCAG 2.2: https://www.w3.org/WAI/WCAG22/quickref/
- React Accessibility: https://react.dev/reference/react-dom/components#form-components

本ファイルには、プロジェクト固有の方針のみ記載する。

---

## 基本方針（プロジェクト固有）

- WCAG 2.2 Level AA準拠を目標
- アクセシビリティは設計段階から考慮

## 必須チェックリスト（プロジェクト固有）

- [ ] すべての画像に適切なalt属性がある
- [ ] 見出しが階層順になっている
- [ ] フォーム要素にラベルが紐付いている
- [ ] キーボードで全機能にアクセスできる
- [ ] フォーカスインジケーターが見える
- [ ] 色のコントラスト比が十分（通常テキスト4.5:1以上）
- [ ] 色だけに依存した情報伝達がない
- [ ] エラーメッセージが明確

## テスト（プロジェクト固有）

- eslint-plugin-jsx-a11yを導入
- キーボードのみで操作できるか手動確認
