# QALinker ビジョンドキュメント

このディレクトリには、QALinkerプロジェクトの根本的な思想、ビジョン、ビジネスモデルに関するドキュメントを格納します。

## ドキュメント構成

- `philosophy.md` - プロジェクトの根本思想と設計哲学
- `business-model.md` - ビジネスモデルと収益化戦略
- `roadmap.md` - フェーズ別ロードマップ
- `knowledge-architecture.md` - ナレッジベースの時間軸設計

## 目的

これらのドキュメントは、開発の方向性を確認するための「北極星」として機能します。
機能追加や設計判断に迷った際は、これらのドキュメントに立ち返り、本来の思想と合致しているかを確認してください。
